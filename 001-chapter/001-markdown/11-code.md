## Код в одну строку

Такой кусочек `var temp = 1 < 2 && 5 > 4;` будет встроен в текст.

## Много строк кода 

Такой кусочек 
```
def hello_world():
    print("Hello, World!")
```

не будет всроен в текст

## Указание языка программирования и частичная подсветка синтаксиса

Пример:

```python
def hello_world():
    print("Hello, World!")
```
